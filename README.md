# My First Rust App

Welcome to my first Rust application! This project is a simple Rust application designed to help me learn the basics of the Rust programming language.

## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- [Rust](https://www.rust-lang.org/tools/install)

### Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/yourusername/my_first_rust_app.git
    ```

2. Navigate to the project directory:

    ```sh
    cd my_first_rust_app
    ```

3. Build the project:

    ```sh
    cargo build
    ```

### Running the Application

To run the application, use the following command:

```sh
cargo run
```
